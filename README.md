# Memory Card Game

A memory card game where players flip cards to find matching pairs.

Two difficulty levels to choose from. Tracks your moves, time and 
best score. The board is built dynamically using JavaScript DOM 
manipulation.

## Built with
- HTML
- CSS
- JavaScript

## Live demo
https://martinlister1.github.io/memory-game

## Run locally
Just open index.html in your browser — no setup needed.
