# Memory Card Game
Built by Martin Lister | HTML, CSS & JavaScript

A browser based memory card game where you flip cards to find matching pairs.

## How to play
1. Click a card to flip it over
2. Click a second card to try and find a match
3. If they match they stay flipped and turn green
4. If they dont match they turn red and flip back over
5. Find all the pairs to win!

## Features
- Easy mode - 8 pairs
- Hard mode - 12 pairs
- Move counter
- Pair counter
- Timer
- Best score tracker - keeps your lowest moves for each difficulty
- Cards turn red when they dont match
- Cards turn green when they match

## How to run
Just open index.html in your browser - no install needed!
